## 2️⃣ فكرة النظام وما هو عمله

النظام مصمم ليكون بسيطًا، عمليًا، وسهل الاستخدام لكل من المستخدم العادي ومدير النظام.

# Hotel Booking System – الهيكلية الكاملة مع الشرح

> هذا المخطط يعرض **كامل هيكلية المشروع في مكان واحد**، وكل ملف أو مجلد أمامه **وظيفته مباشرة** بحيث أي شخص يقرأه يفهم المشروع بسرعة.

```
hotel-booking-system/                 ← المجلد الرئيسي للمشروع
│
├── index.php                          ← الصفحة الرئيسية للموقع، نقطة بداية النظام
├── config.php                         ← تحميل الإعدادات العامة واستدعاء الاتصال بقاعدة البيانات
├── logout.php                         ← تسجيل خروج المستخدم وإنهاء الجلسة
│
├── config/                            ← إعدادات النظام
│   └── database.php                  ← الاتصال بقاعدة بيانات MySQL (host, user, password, db)
│
├── auth/                              ← المصادقة (تسجيل دخول / تسجيل)
│   ├── login.php                     ← تسجيل دخول المستخدم أو المدير وتحديد الصلاحية
│   ├── register.php                  ← إنشاء حساب مستخدم جديد
│   └── register-admin.php            ← إنشاء حساب مدير النظام
│
├── admin/                             ← صفحات لوحة تحكم المدير
│   ├── dashboard.php                 ← لوحة التحكم: إحصائيات المستخدمين والحجوزات
│   ├── users.php                     ← عرض وإدارة جميع المستخدمين
│   ├── user-add.php                  ← إضافة مستخدم جديد يدويًا
│   ├── user-edit.php                 ← تعديل بيانات مستخدم موجود
│   ├── users-export.php              ← تصدير بيانات المستخدمين (Excel / CSV)
│   ├── reports.php                   ← عرض تقارير الحجوزات والاستخدام
│   └── settings.php                  ← إعدادات النظام العامة
│
├── user/                              ← صفحات المستخدم العادي
│   ├── dashboard.php                 ← لوحة المستخدم: ملخص الحجوزات
│   ├── bookings.php                  ← عرض جميع حجوزات المستخدم
│   └── profile.php                   ← تعديل الملف الشخصي وكلمة المرور
│
├── pages/                             ← صفحات التصفح والحجز
│   ├── hotels.php                    ← عرض جميع الفنادق المتاحة
│   ├── hotel-details.php             ← تفاصيل فندق معين والغرف التابعة له
│   ├── booking.php                   ← تنفيذ عملية الحجز (اختيار التواريخ والغرفة)
│   ├── booking-details.php           ← عرض تفاصيل حجز محدد
│   ├── booking-confirmation.php      ← تأكيد نجاح عملية الحجز
│   └── bookings.php                  ← عرض الحجوزات بشكل عام
│
├── api/                               ← واجهات برمجية (AJAX)
│   ├── get-rooms.php                 ← جلب الغرف من قاعدة البيانات بدون إعادة تحميل الصفحة
│   └── check-availability.php        ← التحقق من توفر الغرف ومنع التعارض في الحجوزات
│
├── includes/                          ← ملفات مشتركة تستخدم في جميع الصفحات
│   ├── header.php                    ← رأس الموقع والقائمة العلوية
│   ├── footer.php                    ← تذييل الموقع
│   └── functions.php                 ← دوال مشتركة (جلسات، حساب السعر، التحقق من الدخول)
│
├── css/                               ← تنسيقات الموقع
│   └── style.css                     ← التحكم في الألوان، الخطوط، والتصميم
│
├── js/                                ← الجافاسكربت والتفاعل
│   └── script.js                     ← AJAX، التحقق من النماذج، التحديث بدون Reload
│
└── hotel_booking.sql                  ← قاعدة البيانات (users, hotels, rooms, bookings)
```

## ملخص سريع لآلية العمل

* المستخدم يسجل دخول من `auth/login.php`
* يتصفح الفنادق من `pages/hotels.php`
* يختار غرفة ويحجز من `pages/booking.php`
* البيانات تُحفظ في MySQL
* المدير يراقب ويدير النظام من مجلد `admin/`

> هذه الهيكلية جاهزة للاستخدام في **شرح مشروع، تقرير جامعي، أو توثيق رسمي**.

## 8️⃣ كيفية تشغيل النظام

### 1️⃣ تثبيت XAMPP

* تشغيل Apache
* تشغيل MySQL

### 2️⃣ نسخ المشروع

ضع مجلد المشروع داخل:

```
C:\xampp\htdocs\
```

### 3️⃣ استيراد قاعدة البيانات

* افتح phpMyAdmin
* أنشئ قاعدة بيانات باسم:

```
hotel_booking
```

* استورد ملف `database.sql`

### 4️⃣ إعداد الاتصال بقاعدة البيانات

في ملف `config.php`:

```php
$conn = new mysqli('localhost', 'root', '', 'hotel_booking');
```

### 5️⃣ تشغيل النظام

افتح المتصفح واكتب:

```
http://localhost/hotel-booking-system/
```
